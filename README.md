# 🚀 Dashboard CPE 2026

Dashboard orçamentário do CPE rodando no Render (URL pública permanente).

## ⚡ Setup Rápido (3 passos)

### PASSO 1️⃣ — Cria repo no GitHub

1. Acesse https://github.com/new
2. Nome do repo: `cpe-dashboard`
3. Descrição: `Dashboard Orçamentário CPE`
4. Clique em **"Create repository"**

### PASSO 2️⃣ — Copia os arquivos

1. Baixe estes 5 arquivos:
   - `package.json`
   - `server.js`
   - `dashboard.html`
   - `render.yaml`
   - `.gitignore`

2. No seu repo GitHub (web):
   - Clique em **"Add file"** → **"Upload files"**
   - Selecione os 5 arquivos
   - Clique em **"Commit changes"**

### PASSO 3️⃣ — Conecta no Render

1. Acesse https://render.com
2. Clique em **"New"** → **"Web Service"**
3. Clique em **"Connect repository"**
4. Selecione seu repo `cpe-dashboard`
5. Deixe as configurações padrão
6. Clique em **"Deploy"**

**Pronto!** ✅ Render gera uma URL tipo: `https://cpe-dashboard.onrender.com`

---

## 🎯 Como usar

- **Abrir dashboard:** Clique na URL que o Render gerou
- **Atualizar dados:** Clique no botão "🔄 Atualizar Dados"
- **Editar código:** Faça upload de novos arquivos no GitHub, Render atualiza automaticamente
